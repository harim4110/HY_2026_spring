# Cognitive Tasks Demo

Simple classroom web app for two behavioral tasks:

- Stroop
- Visual Search
- Brain State ROI Viewer

## Features

- Mobile-friendly static site
- Google Sheets auto-submission via Apps Script
- Student ID based submissions with automatic attempt numbering
- Separate results page for instructor use
- Interactive 3D brain template with ROI activation presets for classroom discussion

## Pages

- Student page: `/`
- Results page: `/results.html`
- Brain ROI study page: `/brain-study.html`
- Brain quiz page: `/brain-viewer.html`
- Brain admin answer key: `/brain-viewer-admin.html`
